## /[owner]/[title]/api/v1/

Used for creating simulations.

Supports POST HTTP actions.

### Create simulation

```bash
POST hdoupe/Matchups/api/v1/
```

**Example:**

```json
{
  "meta_parameters": {
    "use_full_data": true
  },
  "adjustment": {
    "matchup": {
      "pitcher": "Max Scherzer"
    }
  }
}
```

**Response:**

```bash
HTTP/1.1 201 Created
Allow: GET, POST, HEAD, OPTIONS
```
